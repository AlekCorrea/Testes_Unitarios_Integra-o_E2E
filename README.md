# Projeto de Testes - Desenvolvimento Web

## 📚 Descrição

Este repositório foi desenvolvido como parte da disciplina de Desenvolvimento Web, com o objetivo de aplicar conceitos de qualidade de software e testes automatizados em um projeto prático.

## 🧪 Testes Implementados

O projeto utiliza Jest para execução dos testes automatizados.

Atualmente, estão implementados:

- Testes unitários

Os testes de integração e end-to-end ainda podem ser implementados futuramente.
